# A 股 AKShare 量化工具实现计划

## 范围

本次实现一个可在当前项目直接运行、后续可迁移为 Codex skill 的 A 股量化分析工具。范围包括多因子选股、个股诊断、大盘与板块概览、单股策略回测四个 CLI 入口。

## 设计原则

- 数据源通过 AKShare 获取，但输出必须保留数据时间、抓取时间、接口名、warning 和 error，避免把公开网页数据包装成确定可靠的商业行情。
- 诊断输出技术信号和观察点，不输出买卖指令。
- 回测显式写清复权、手续费、滑点、下一交易日成交等假设，避免虚假严谨。
- 核心计算逻辑与 AKShare 抓取逻辑分离，使单元测试不依赖网络。
